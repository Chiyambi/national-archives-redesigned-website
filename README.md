# National Archives Redesigned Website

A modern redesign of the National Archives website featuring improved user interface and navigation.

## Overview

This project is a comprehensive redesign of the National Archives web portal, featuring an updated look, improved accessibility, and better user experience.

## Project Structure

```
├── index.html              # Main homepage
├── application.html        # Application page
├── information.html        # Information page
├── laws.html              # Laws and regulations page
├── overview.html          # Overview page
├── preface.html           # Preface page
├── rules.html             # Rules page
├── search.html            # Search page
├── staff.html             # Staff page
├── biblio_preface.html    # Bibliography preface
├── biblio-contents.html   # Bibliography contents
├── archives-styles.css    # Main stylesheet
├── emblem.jpg            # National Archives emblem
├── images-25.jpeg        # Additional images
└── prefecase             # Additional resources
```

## Features

- Responsive design
- Clean and modern interface
- Easy navigation
- Comprehensive content organization
- Professional styling

## Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/Chiyambi/national-archives-redesigned-website.git
   ```

2. Navigate to the project directory:
   ```bash
   cd national-archives-redesigned-website
   ```

3. Open `index.html` in your web browser to view the site.

## Files

- **HTML Files**: Contains the structure and content of each page
- **CSS File**: `archives-styles.css` - Contains all styling for the website
- **Images**: Digital assets used throughout the site

## Technologies Used

- HTML5
